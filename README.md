# SportZone Ecommerce 

**SportZone** is a modern, responsive, and feature-rich E-commerce platform dedicated to sports gear, jerseys, and equipment. Built with pure web technologies to ensure maximum performance and a smooth user experience.

---

##  Key Features

-   **Dynamic Image Slider**: Home page features an interactive, autoplay slider for featured products.
-   **Advanced Filtering & Sorting**:
    -   Filter products by Category (Shoes, Balls, Clothing).
    -   Filter by Size (S, M, L, XL, Standard).
    -   Sort by Price (Low to High, High to Low).
-   **Full Shopping Cart System**:
    -   Persistent cart using `LocalStorage`.
    -   Quantity controls (increase/decrease).
    -   Real-time cart badge updates.
-   **User Authentication**:
    -   Secure Register and Login system.
    -   Session persistence using `LocalStorage`.
-   **Product Details**: Dynamic detail pages for every product in the inventory.
-   **Responsive Design**: Fully optimized for mobile, tablet, and desktop screens.
-   **Clean Codebase**: Organized into components, models, and utility modules using ES6+ Modules.

## 🚀 Technologies Used

-   **HTML5**: Semantic structure.
-   **CSS3**: Custom styling with variables and modern layouts (Flexbox/Grid).
-   **JavaScript (ES6+)**: Custom logic, modular architecture, and DOM manipulation.
-   **FontAwesome**: Modern iconography.
-   **Google Fonts**: "Inter" for a clean, premium typography feel.

##  Project Structure

```text
SportZone/
├── css/            # Stylesheets
├── html/           # All HTML pages
├── js/
│   ├── components/ # Reusable UI components (Navbar, Slider, Footer)
│   ├── models/     # Data models (Product, Cart, User)
│   ├── utils/      # Utility helpers
│   └── ...         # Page-specific scripts
├── products.json   # Product database
└── index.html      # Entry point (Redirects to html/register.html)
```
